# Changelog

## [1.0.0] - 2025-02-11

### Added
- Initial release
- Deterministic SVG avatar generation from any string
- 13 feature layers: head shape, spikes/horns, eyes, eyebrows, mouth, nose, cheeks, ears, face markings, head accessories, belly patch, tail, background pattern
- 1.7B+ unique combinations
- 16 color palettes
- Web component `<avatar-us>` with Shadow DOM
- Hover animation on web component (CSS-only, disable with `no-hover`)
- Cloudflare Worker for image URL API
- `gradient` and `solid` variants
- Optional initial letter overlay
- TypeScript type definitions
- Zero runtime dependencies
