# Visual Style Proposals (v2)

Three face-based avatar styles with modern muted palettes. All clearly recognizable as faces suitable for default profile pictures on any SaaS product.

## C-v2: "Minimal Face"
Clean, simple faces. Improved proportions, muted palette, cheek blush.

### Palette
All use modern muted colors (slate/stone/sage/taupe/dusty rose). No bright primaries. Face: warm off-white. Features: near-black. Professional and suitable for any SaaS product.
